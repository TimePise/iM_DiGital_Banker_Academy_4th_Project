# 📍 추천시스템 기반 기업금융 전략 도출  
### iM DiGital Banker Academy 6기 정형데이터 기반 ML & DL 프로젝트

---

## 1. 프로젝트 개요 (Overview)

본 프로젝트는 **iM뱅크 내부 B2B 기업금융 거래 데이터**를 활용하여  
전국 **시군구 단위의 기업금융 수요 구조를 벡터화**하고,  
**추천시스템 기반 지역 유사도 분석**을 통해  
👉 **신규 영업점 출점에 적합한 유망 지역을 데이터 기반으로 추천**하는 것을 목표로 합니다.

기존의 영업점 입지 선정이 경험과 직관에 의존하던 방식에서 벗어나,  
**정량적·재현 가능한 추천 프레임워크**를 구축함으로써  
시중은행 전환 이후 iM뱅크의 **전국 단위 영업망 확장 전략**을 지원하고자 합니다.

---

## 2. 프로젝트 배경 및 문제 정의 (Background & Problem)

- iM뱅크는 지방은행에서 **시중은행으로 전환**하며 전국 단위 영업망 확대를 추진 중
- 그러나 기존 영업점 입지 전략은  
  - 일부 지역에 편중  
  - 경험·직관 중심 의사결정  
  - 전국 단위 비교 가능한 기준 부재  
  라는 한계를 가짐
- 내부적으로는 예금·여신·카드·외환 등 **풍부한 B2B 거래 데이터**를 보유하고 있으나  
  👉 이를 **지역 단위로 통합·비교·추천하는 체계가 부재**

### ❓ 핵심 질문
> “기존에 성과가 검증된 신규 지점 지역과 **금융 구조가 유사한 다른 유망 지역은 어디인가?**”

---

## 3. 프로젝트 목표 (Objectives)

1. **시군구 단위 기업금융 데이터 벡터화**
   - 예금, 여신, 카드, 외환, 채널 이용 구조 등 다차원 특성 생성
2. **지역 간 유사도 기반 추천 시스템 구축**
   - 금융 패턴 중심의 지역 유사도 산출
3. **신규 영업점 후보 지역 Top-N 추천**
   - 기준 지역과 유사한 산업·금융 구조를 가진 지역 도출
4. **실무 적용 가능한 전략 인사이트 도출**
   - 제조업 중심, 신도시형, 복합 산업 도시 등 유형별 해석

## 4. 사용 데이터 (Dataset)

### 🔹 내부 데이터 (iM뱅크 기업금융)
- 기간: **2022.01 ~ 2024.12 (36개월)**
- 단위: **법인 기업 × 월**
- 주요 변수:
  - 예금: 요구불/거치식/적립식/신탁/퇴직연금
  - 여신: 운전자금/시설자금 대출
  - 거래: 카드·외환·채널별 거래금액 및 건수
  - 구조 지표: 여신비율, 디지털거래비중, 카드거래비중
  - 관계 지표: 전담 RM 여부, 기업등급

### 🔹 외부 데이터
- **전국 시군구별 월별 인구수 (2024년)**  
→ 지역 규모 및 성장성 보조 지표로 활용

## 5. 분석 및 모델링 파이프라인 (Methodology)

### ① 데이터 전처리
- 로그 변환 (규모 변수)
- 다중공선성 제거 (VIF < 10 기준)
- **RobustScaler 적용**
  - 이상치 영향 최소화
  - 금융 데이터 특성 반영

### ② 1단계: Cosine Similarity 기반 유사도 (Baseline)
- 지역별 금융 특성 벡터 생성
- **93 × 93 지역 유사도 행렬** 도출
- 금융 패턴의 **방향성 유사성** 중심 비교

### ③ 2단계: ALS 임베딩 (실험)
- 지역 × 업종 사업체 수 피벗
- Implicit ALS로 **잠재 요인(30차원)** 학습
- 업종 구조 기반 유사도 산출 (85 × 85)

### ④ 3단계: 하이브리드 결합
- Cosine + ALS 가중 평균
- Alpha 튜닝 (Top-k Jaccard Similarity)
- 결과적으로 **Cosine 단독 모델이 가장 안정적**임을 확인

## 6. 기준 지역 및 추천 결과 (Key Results)

### 📌 기준 지역
- 경기도 화성시
- 서울특별시 금천구
- 서울특별시 강서구
- 충청남도 천안시 서북구  
(최근 iM뱅크 신규 출점 지역)

### 📌 추천 결과 요약
- **화성시 → 양산·김해·창녕**
  - 제조업·산업단지 중심 기업금융 수요
- **금천구 → 부산 사상·평택**
  - 중소기업·도심형 복합 산업 구조
- **강서구 → 울주·영등포·인천 서구**
  - 물류·주거·첨단산업 복합 지역
- **천안 서북구 → 기장·파주·하남**
  - 신도시·상권·산업 동시 성장 지역

👉 단순 지리적 인접성이 아닌  
**산업 구조 + 금융 수요 패턴 유사성** 기반 추천임을 확인

## 7. 프로젝트 인사이트 (Insights)

- 금융 데이터 기반 지역 유사도는  
  **신규 지점의 조기 수익화 가능성(BEP 단축)** 판단에 활용 가능
- 제조업·신도시·복합 산업 지역별로  
  **서로 다른 기업금융 전략 수립 필요**
- 데이터 기반 입지 선정은  
  **NIM 하락 환경에서 비이자·기업금융 확대 전략**으로 연결 가능
